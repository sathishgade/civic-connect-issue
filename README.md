# Civic Connect

A smart civic issue reporting platform designed to empower citizens and streamline city administration.

## Features

-   **Citizen Dashboard**: Report issues, track status, and view history.
-   **Admin Dashboard**: Manage complaints, assign tasks, and manage users.
-   **Employee Dashboard**: View assigned tasks and mark them as resolved.
-   **Voice Reporting**: Multilingual AI voice assistant for easy reporting.
-   **QR Verification**: "Proof of Presence" system for task validation.

## Tech Stack

-   **Frontend**: React, TypeScript, Vite, Tailwind CSS, Shadcn UI
-   **Backend**: Firebase (Firestore, Auth)
-   **AI/ML**: Speech Recognition, Image Analysis (Planned)

## Getting Started

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Run Development Server**:
    ```bash
    npm run dev
    ```

3.  **Build for Production**:
    ```bash
    npm run build
    ```
